# Developer Portfolio Website

A responsive **personal developer portfolio website** built using **HTML, CSS, and JavaScript**. This project showcases a developer's profile, skills, experience, and projects in a clean and modern interface.

---
## 🚀 Features

* Responsive and modern UI design
* Smooth navigation with anchor links
* Animated typing effect for job roles
* About section with skills list
* Work experience timeline
* Contact form with validation
* Smooth scrolling behavior

---
## 🛠️ Technologies Used

* **HTML5** – Structure and content of the website
* **CSS3** – Styling and layout design
* **JavaScript (Vanilla JS)** – Interactivity and form validation
* **Google Fonts (Poppins)** – Typography
* **Typed.js** – Typing animation effect

---
## 📄 Sections Included

### 1. Home

* Introduction section with a greeting
* Animated typing text showing roles such as **Web Developer** and **Web Designer**

### 2. About

* Brief introduction about the developer
* Skills list including:

  * HTML
  * CSS
  * JavaScript
  * React
  * Node.js
  * Git

### 3. Experience

* Timeline-style layout showing learning and work experience in technologies such as:

  * HTML
  * Node.js
  * React

### 4. Projects

* Section reserved to showcase developer projects.

### 5. Contact

* Contact form with client-side validation for:

  * Name
  * Email
  * Message


---
## ⚙️ JavaScript Functionality

### Typing Animation

Uses **Typed.js** to create a typing animation effect:


var typed = new Typed('#element', {
    strings: ['Web Developer', 'Web Designer!'],
    typeSpeed: 100,
});

### Form Validation

The contact form validates:

* Empty fields
* Valid email format

If validation passes, a confirmation message is shown.
